# Code of Conduct

## Our standard

This project is maintained as a focused, professional open-source tool. Contributions and discussions should be technical, constructive, and respectful.

**Expected behavior:**
- Be direct and technical — this is a small project, not a community forum
- Critique code and ideas, not people
- Accept that maintainers have final say on direction and design decisions

**Not acceptable:**
- Personal attacks, harassment, or discriminatory language
- Spam, off-topic promotion, or low-effort noise
- Publishing others' private information

## Enforcement

Violations can be reported to [sebastian@wiar8.com](mailto:sebastian@wiar8.com). Responses will be proportionate — from a warning to a permanent ban depending on severity.

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions, and any other channel where you represent this project.
